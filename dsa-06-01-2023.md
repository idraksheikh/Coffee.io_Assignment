### [Maximum Ice Cream Bars](https://leetcode.com/problems/maximum-ice-cream-bars/)

## My Approach 

Sorted the array an then count the Icream we can by with the coins.

```java

class Solution {
    public int maxIceCream(int[] costs, int coins) {
        int i,count,n;
        i=count=0;
        n=costs.length;
        Arrays.sort(costs);
        while((i<n)&&(coins>0)){
            if(coins>=costs[i]){
                coins-=costs[i];
                count++;
            }
            i++;
        }
        return count;
    }
}

```
